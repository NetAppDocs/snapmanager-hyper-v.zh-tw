= 執行災難恢復
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
災難恢復功能可確保如果發生災難或其他情況、導致主要儲存站台的關鍵受保護資料無法使用、您可以透過次要儲存站台存取該資料的備份複本。災難恢復只能使用PowerShell介面執行。
