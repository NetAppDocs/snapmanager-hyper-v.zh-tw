= 從非ONTAP主機刪除先前匯出的VM
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
成功匯入虛擬機器（VM）之後、即可從非ONTAP主機刪除VM。

VM必須已成功匯入。

. 選取原本裝載VM的非ONTAP主機。
. 在「虛擬機器」窗格中、以滑鼠右鍵按一下先前匯出的VM名稱。
. 按一下*刪除*。
+
隨即開啟視窗、要求您確認是否要刪除VM。

. 按一下*刪除*。
+
VM會從其原始主機移除。
