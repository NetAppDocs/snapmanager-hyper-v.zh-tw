= 用於無聲安裝的MSI自訂屬性
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


* *問題*
+
執行Windows Server 2008或Windows Installer 4.5版的系統無法辨識SnapManager 出適用於Hyper-V安裝的內建功能。

* *修正行動*
+
安裝時使用MSIRESTARTMANDERCONTROL=停用命令切換參數。
