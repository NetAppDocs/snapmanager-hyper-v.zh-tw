= 檢視Hyper-V父主機或主機叢集
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以檢視特定Hyper-V父主機或主機叢集的組態資訊、以便監控其狀態。

. 在導覽窗格中、按一下* Protection（保護）*>* hosts*。
. 選取您要檢視的主機或主機叢集。
+
「Details（詳細資料）」窗格會顯示主機或主機叢集名稱、網域、叢集成員（若適用）和組態訊息。如果您選取未設定的主機、「詳細資料」窗格會顯示尚未設定的項目相關資訊。
