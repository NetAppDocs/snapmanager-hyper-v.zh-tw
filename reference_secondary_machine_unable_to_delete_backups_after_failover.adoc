= 在容錯移轉後刪除備份
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


* *訊息*
+
資料集中的某些物件不存在指定的備份。

* *說明*
+
容錯移轉至次要站台（站台B）之後、您可能無法刪除在主要站台（站台A）建立的備份。如果您位於災難恢復站台（站台B）、並嘗試刪除在主要站台（站台A）所做的備份、您將會從主要站台（站台A）刪除備份、而非從災難恢復站台（站台B）刪除備份。

* *修正行動*
+
執行災難恢復作業之後、只會刪除您目前營運站台所做的備份。
