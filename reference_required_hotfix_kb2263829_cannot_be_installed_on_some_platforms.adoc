= 某些平台無法安裝必要的修補程式KB2263829
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


* *問題*
+
安裝SnapManager 適用於Hyper-V的功能時、嘗試安裝Windows Server 2008 R2 SP1的修補程式KB2263829可能會失敗。安裝程式指出此修補程式不適用於此伺服器。

* *原因*
+
您的平台不支援此修補程式。

* *修正行動*
+
與Microsoft一起開啟支援案例、並與Microsoft一起解決問題。
