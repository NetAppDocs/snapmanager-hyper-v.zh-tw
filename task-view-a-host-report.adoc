= 檢視主機報告
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以檢視SnapManager 有關以支援Hyper-V的方式管理主機的報告

.步驟
. 在導覽窗格中、按一下功能表：Reports[host]。
. 選取包含您要檢視之報告的主機。
. 從「報告」窗格中、選取您要檢視的報告、然後按一下「*檢視報告*」。
+
組態報告會出現在另一個視窗中。
