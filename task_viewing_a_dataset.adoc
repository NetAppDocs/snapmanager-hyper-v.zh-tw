= 檢視資料集
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以檢視與資料集相關聯的虛擬機器。

. 在導覽窗格中、按一下* Protection（保護）*>* Dataset *。
. 展開樹狀檢視、查看屬於資料集的虛擬機器。
